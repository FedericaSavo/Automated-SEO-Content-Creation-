## Overview

This project automates the creation of SEO and marketing content. Using a daily workflow in **n8n**, it reads a list of topics from a Google Sheet, generates content for each topic (blog posts, marketing snippets, SEO-focused text) and publishes a Notion page for every topic.

---

## Features
- Daily automation via n8n workflow.
- Read topics from Google Sheets.
- Automatically generate content for each topic using AI tools or custom scripts.
- Create and publish pages in Notion for each topic.
- Ideal for SEO blogs, content marketing and consistent content pipelines.

---

## Technologies Used
- **n8n** – automation workflow engine
- **Google Sheets API** – source of topics
- **Notion API** – publish generated content

---

## How It Works
1. **Trigger**: The workflow runs daily via n8n scheduler.  
2. **Read Topics**: Pull a list of topics from a Google Sheet.  
3. **Content Generation**: For each topic, generate text content suitable for SEO or marketing purposes.  
4. **Publish to Notion**: Automatically create a new Notion page for each topic with the generated content.
